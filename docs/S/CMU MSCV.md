# CMU MSCV


## 项目介绍
SCS Robotics Institute下面1.5年的CV项目，强制暑期实习但也可以通过校内暑研来代替


## 录取偏好
很看重是否match，喜欢有cv顶会与科研/cv实习或工作经验，没有cv bg需要三维很高。

大部分为陆本/港台本和印度人，陆本招收背景比较广泛，印度同学基本都是有大约两年的工作经验
