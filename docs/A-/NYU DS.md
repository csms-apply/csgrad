# NYU DS

## 项目介绍
很适合cs/ds/math本科的同学来这里读MS然后增强research bg然后转phd的，可以去ins关注
nyucds，可以看到它们很重视research。不怎么适合转码，适合找ds工作。
有不少在本系转博的，找research的同学，录取后就可以去跟教授发邮件了，很多项目summer就可以进组了



课程硬核，对数学要求比较高，因此招数学系的同学很多。来的同学基本都是为了读博/找ds工作的，安心学两年学到的东西基本能匹配任何一家ds岗位了。（不过在LLM横行的年代，ds也都得会ml/dl等必备技能了），能很好的学习到当一名Data Scientist所需要的知识,career fair来的比较多的，也都是纽约金融业的公司比较多
