# Upenn MSR

## 项目介绍
来之前对这个program的想象是“很硬核、很卷、实验室卷到飞起”……
结果来了之后才发现：ROBO真的是全美最自由、最能“do yourself”的机器人硕士项目之一！ 并且也可以dual mcit/mcis等

一句话总结我的最大Takeaway
“高度自由 = 无限可能，也意味着你必须非常清楚自己想要什么”
没有既定路线，想学什么就学什么，想做哪个方向的研究就去找对应大牛，真的可以完全按照自己的兴趣和职业规划来塑造这份硕士经历。

费城消费：中等偏高，低于纽约/湾区，但高于休斯敦/中部城市。
房租（合租单间）：$900-1400/月；吃饭自己做+偶尔外食：$500-800/月。平时作息：
不做research：周一到周五上课+写作业，效率高的话还有很多空闲时间参加各种event。
做research：上课+作业+lab，一周research大概10-20小时，时间会比较满。
但Penn Engineering每周几乎都有各种research seminar，对方向感兴趣的同学可以把空闲时间塞满～

## 录取bar 和dp
每年秋季入学新生大约30人左右，中国人通常6-8人。
背景非常多样：有国内顶尖本科直申的，有美本转过来的，也有工作几年再来深造的。
录取确实competitive，但有个神奇的“后门”——如果你先进Penn Engineering其他项目（比如EE,SE），对robotics真感兴趣，转ROBO其实相对容易很多，很多学长姐都是这么操作的。

1. umass 机械工程本科，有多段机器人相关实习
2.  Worcester Polytechnic Institute 机器人本科gpa3.95
3. iit 自动化本科gpa 9.04/10, 三年全职经验
4. 印度本科gpa 8.72/10
5. nyu cs+ds gpa 3.87/4
## 项目找工

最多的方向还是Computer Vision + Coding相关岗位（FAANG/大厂AI部门、自动驾驶、机器人公司等），也有不少去Control、Planning、Manipulation方向的。

公司类型：Tesla、Waymo、Aurora、Boston Dynamics、Amazon Robotics、Apple、Meta Reality Labs、Google X 等，title多为Robotics Engineer、Perception Engineer、Software Engineer - Robotics等。

工资水平（参考前几年数据）：起薪一般在15-22万美元之间（base + bonus + RSU），看公司和地点差异较大。

1. upenn ee本科，上岸小厂硬件实习
2. upenn ce本科，上岸amazon 硬件实习
3. iit 自动化本科gpa 9.04/10, 三年全职经验，上岸amazon ds
4. 印度本科gpa 8.72/10，多段实习，上岸sf 小厂cv engineer
4. upenn ee 本科，有实习经历，上岸space x sde test intern
5. nyu cs+ds 本科，有小厂ds intern，上岸amazon ds 实习
## 大牛教授

GRASP lab全美数一数二， 
随便举几个响当当的名字：
1. Vijay Kumar（无人机、空中机器人之神）
2. George Pappas（控制与优化大神）
3. Mark Yim（模块化机器人传奇）
4. Kostas Daniilidis（计算机视觉+机器人感知）
5. Jianbo Shi（CV经典人物）

这些教授手下几乎每年都有很多master/PhD在做research，机会非常多。
只要你方向匹配、主动沟通，找到带你的PhD带你入lab的概率很高。
