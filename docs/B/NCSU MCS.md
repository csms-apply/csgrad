# NCSU MCS

## 项目介绍
可以转码，找工不错，北卡是个好地方，旁边有很多大厂。
项目workload总的来说相比地里的大多数热门CS项目来说不大，有较多时间刷题&投简历



## 录取bar和dp
很友好，可以保底，3.2+托福100就行

大部分人是985转码和211科班，也有少数美本
