# CMU MSAII

## Program Overview
Flagship AI program under SCS, career-oriented. If you want to do research, look at SCS MSR or MSML instead. The program is very flexible and career-change friendly, with plenty of career-change data points from all kinds of backgrounds. The department head prefers candidates with strong metrics -- the higher the GPA, GRE, and TOEFL the better (TOEFL 105+, GRE 330+, top GPA). There are many low-workload easy courses, and the 8 elective slots can all be filled with hardcore SCS systems courses -- CC, ACC, OS, Storage are all available. There's a Coding Bootcamp LeetCode grinding course, which is very friendly for finding SDE positions.

Internship companies basically cover FAANG. SDE and MLE are the main career paths, with nearly 100% employment rate. Almost nobody pursues a PhD.
